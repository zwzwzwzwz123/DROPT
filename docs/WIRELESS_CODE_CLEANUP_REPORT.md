**详细说明**:

#### **`env/env.py` - 无线网络功率分配环境**
- **功能**: 实现了AIGC网络优化环境
- **核心概念**: 
  - 信道增益（好信道 vs 差信道）
  - 功率分配（10个正交信道）
  - 速率计算（Shannon公式）
  - 水注入算法（最优功率分配）
- **状态空间**: 信道增益 + 奖励（11维）
- **动作空间**: 功率分配方案（10维）

#### **`env/utility.py` - 水注入算法和信道模型**
- **功能**: 无线通信专用算法和工具函数
- **核心算法**:
  - `water()`: 水注入算法（Waterfilling Algorithm）
  - `rayleigh_channel_gain()`: 瑞利信道增益生成
  - `CompUtility`: 奖励计算类
- **数学模型**:
  - Shannon容量公式: C = log₂(1 + g·p)
  - 水注入最优解: p_n = max(α - N₀/g_n, 0)
  - 瑞利衰落模型

**删除理由**: 这些文件完全针对无线网络场景，与数据中心HVAC优化无关。
